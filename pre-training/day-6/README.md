## Day-6

Forward Propagation: is like a one way output lane where the enter an input, that input is inside a hidden layered process and then an output is produced.

What would need to happen for this network to actually learn something? (What's Missing)

Whats missing is the feedback system, where we can correct the output and then adjust the weights and biases to get a better output next time. This would come under the category of back propogation, through which we can adjust weights, and biases to get a better output.
